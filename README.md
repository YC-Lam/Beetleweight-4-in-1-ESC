# beetleweight 4 in 1 brushless ESC
 a 4 in 1 ESC designed for UK beetleweight robots. It can power two brushless drive and one weapon motor.

## Features
- Two 20A brushless drive motor
- Single 80A brushless weapon motor
- Integrated fuse to accommodate BBB rules
- On-board gyroscope to invert controls when bot is inverted (optional)
- On-board BLE reciever to control bot over an XBox controller
- Supports 2-6s Lipo battery
- configurable UVLO, OVP and OCP
- Compact with a 50x50mm dimension
- Single side SMD soldering for easy manufaturing (some hand soldered resistors at the back)
